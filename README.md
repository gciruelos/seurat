seurat
======



Supported formats
-----------------
* jpeg, png, bmp, hdr, psd, tga, gif, pic


Example of use
--------------

![example of use](doc/show.gif "Example")


Try it out
----------

Clone the repo in your favourite folder

```sh
git clone https://github.com/gciruelos/seurat.git
cd seurat
```

Compile

```sh
make
```

Use

```sh
./seurat <args>
```
   


Commands
--------

|     command         |  key  | alternative |
|---------------------|:-----:|:-----------:|
| zoom in             |+      |      Z      |
| zoom out            |-      |      z      |
| move                |arrows |             |
| image info          |i      |             |
| change color scheme |c      |      C      |
| change dithering    |d      |             |
| quit                |q      |             |
